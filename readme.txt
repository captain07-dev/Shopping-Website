Instructions

1. Install xampp server
2. extract the zip file and paste the files to the location(your xampp server location/htdocs/paste here) 
   example: C:\xampp\htdocs
3. now open the xampp server from start menu and start the apache and mysql
4. now open your browser(recommended: Google Chrome)
5. copy and paste this url to your browser (http://localhost/phpmyadmin/)
6. make new database in the phpmyadmin(Database name must be named as "shop")
7. now go to import in phpmyadmin and import the file which is given in the zip file(it will take 1-5 minutes)
8. All set!! Hoooraayyy!!!
9. Now you can open your ecommerce site in the browser
   example:(http://localhost/shoppingritik/)
